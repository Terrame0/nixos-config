### уведомления
- нет центра уведомлений
- уведомления не стилизованы

### блокировка и сон
- нет hyprlock — экран не блокируется
- нет hypridle — экран не гаснет, ноутбук не засыпает по бездействию
- `$exit` ссылается на `hyprshutdown`, которого нет в конфиге (падает на fallback `hyprctl dispatch exit`)

### рабочий стол
- анимация изменения размера окна ломается при использовании клавиатуры
- нет индикатора раскладки клавиатуры
- плохо стилизованный лаунчер приложений
- нет меню/апплета NetworkManager — сеть видно в waybar, но переключать сети можно только через CLI
- нет power-меню / logout (wlogout или аналог) — `$exit` просто убивает сессию
- нет OSD громкости/яркости — при нажатии медиа/яркость-клавиш нет всплывашки-индикатора

### вид и интеграция
- нет Qt-темы — qbittorrent и другие Qt-приложения вне общего стиля (нет qt.enable / QT_QPA_PLATFORMTHEME)
- нет xdg-desktop-portal-gtk — установлен только hyprland-портал, при этом в hyprland.conf есть window-rule на класс xdg-desktop-portal-gtk для file-picker
- cursorTheme не задан в Home Manager — курсор работает только через XCURSOR_THEME env, настройки HM нет

### поведение
- нет переключателя тачпада
- playerctl используется в биндах XF86Audio(Next|Play|Prev), но не установлен — медиа-клавиши молча не работают
- wofi запускается через `flock` — костыль против двойного запуска

### питание
- нет power-profiles-daemon — управления профилями питания ноутбука нет (при том что asusd для вентиляторов настроен)
